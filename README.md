# TayoSmart Dashboard

An industry-ready IoT Fire Fighting Monitoring Platform built with modern web technologies.

## Overview

TayoSmart Dashboard is a commercial-grade monitoring platform designed to provide real-time visibility into fire protection systems. It is built to monitor pumps, tanks, pressure, alarms, and other critical infrastructure while supporting future IoT device integration through MQTT and cloud deployment.

## Key Features

* Real-time dashboard
* Fire pump monitoring
* Water tank monitoring
* Alarm management
* Reports and analytics
* Responsive user interface
* Future MQTT integration
* Future AWS deployment
* WebSocket-based live updates
